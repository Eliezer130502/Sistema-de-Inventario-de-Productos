README - Sistema de Inventario de Productos
Descripción del Proyecto
Este proyecto es una aplicación Full Stack para gestionar un sistema de inventario de productos. Permite a los usuarios agregar, ver, editar y eliminar productos, así como cambiar su estado de disponibilidad.

Tecnologías Utilizadas
Backend: Java 17, Spring Framework
Frontend: React
Base de Datos: H2 (en memoria)
Funcionalidades
Backend (5 puntos)
Gestión de productos: agregar, editar, eliminar y listar productos.
Almacenamiento temporal en H2 para simplificar el despliegue.
Frontend (4 puntos)
Interfaz intuitiva para la gestión de productos.
Formulario para agregar y editar productos, mostrando el estado de disponibilidad.
Comunicación entre Frontend y Backend (1 punto)
API RESTful para la comunicación entre el frontend y el backend.
Instrucciones de Uso
Clonar el repositorio:
bash
Copiar código
git clone <URL del repositorio>
Ejecutar el Backend:
Navegar al directorio del backend y ejecutar:
arduino
Copiar código
./mvnw spring-boot:run
Ejecutar el Frontend:
Navegar al directorio del frontend y ejecutar:
sql
Copiar código
